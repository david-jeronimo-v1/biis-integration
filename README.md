# biis-integration

Home MTA 

## Usage

Install Leiningen
On Windows:
* Download lein.bat: https://raw.githubusercontent.com/technomancy/leiningen/stable/bin/lein.bat
* Add it to path

Config: [config.clj](src/config.clj)

Run:
* lein.bat run

Output: Under 'location' folder. One folder created for each policy code
